**Какие интерактивности добавлены на страницу?**

На страницу добавлены четыре полностью рабочие интерактивные фичи:
- Кнопка «Наверх» — появляется при прокрутке вниз и плавно возвращает в начало страницы.
- Аккордеон (FAQ) — четыре вопроса, при открытии одного остальные автоматически закрываются.
- Фотогалерея с фильтрацией — кнопки «Все / Природа / Город / Пляж», изображения показываются/скрываются мгновенно.
- Модальное окно — при клике на любое фото открывается большая версия, закрывается по крестику, по тёмному фону и по клавише Esc.

**Где использовался ИИ, а где код писался вручную?**

Почти весь JavaScript я получил от ИИ (Grok 4).
Я задавала ему отдельные точные запросы и получала готовые куски кода.
Вручную я:
1) объединила все части в один script.js,
2) исправила одну синтаксическую ошибку в аккордеоне (ошибка с &&),
3) написала и подогнала все CSS-стили под свой дизайн,
4) расставила z-index, тестировала в формате мобильного телефона,
5) добавила/убрала тестовую высоту страницы, чтобы проверять кнопку «Наверх».

**Что показалось самым интересным или сложным?**

Самое интересное — как быстро и красиво всё заработало благодаря ИИ. Особенно круто выглядит модальное окно и плавное появление кнопки «Наверх».
Самое сложное — поймать момент, когда из-за одной маленькой ошибки в аккордеоне переставал работать вообще весь скрипт (в том числе кнопка «Наверх»). Ещё немного повозился с z-index и отступами на мобильных, но в итоге всё идеально.

**Какие запросы делали LLM?**

«Напиши JavaScript-код для кнопки "Наверх": круглая, фиксированная в правом нижнем углу, появляется при прокрутке вниз, при клике — плавная прокрутка наверх, без jQuery»

«Сделай аккордеон на чистом JavaScript: несколько пунктов, при открытии одного остальные автоматически закрываются»

«JavaScript-фильтрация изображений в галерее по категориям через data-category и кнопки с data-filter, чтобы при нажатии показывались только нужные фото»

«Модальное окно для увеличения фото: при клике на картинку в галерее открывается полноэкранная модалка с этим фото, закрывается по крестику, по клику на тёмный фон и по клавише Escape»
